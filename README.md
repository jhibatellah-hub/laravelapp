# Cabinet Médical - Application Laravel

Application web de gestion de cabinet médical développée avec le framework Laravel.

## Fonctionnalités

- **Gestion des patients** : Ajout, modification, suppression et consultation des dossiers patients
- **Gestion des rendez-vous** : Planification et suivi des consultations
- **Recherche dynamique** : Recherche en temps réel des patients via AJAX (Axios)
- **Authentification** : Système de connexion sécurisé pour les utilisateurs
- **Tableau de bord** : Vue d'ensemble des activités du cabinet

## Technologies Utilisées

- **Backend** : PHP 8.x, Laravel 10.x
- **Frontend** : HTML5, CSS3, JavaScript, Bootstrap
- **Base de données** : MySQL / MariaDB
- **Requêtes HTTP** : Axios
- **Gestion des dépendances** : Composer, NPM

## Prérequis

Avant d'installer le projet, assurez-vous d'avoir les éléments suivants installés sur votre machine :

- PHP &gt;= 8.1
- Composer
- Node.js & NPM
- MySQL ou MariaDB
- Serveur web (Apache/Nginx) ou Laravel Sail
